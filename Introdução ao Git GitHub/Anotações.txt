O Git Bash é um terminal extendido para otimizar o uso do Git

Comandos Básicos:
git init - para iniciar a pasta
git status - para ver status da pasta
git add -a ou git add . para não stage para stage 
git commit -m "Atualização do Links Úteis do Readme"
git push origin main para subir arquivos para GitHub
git pull origin main para descarregar arquivos do GitHub para pasta
